# Monitoring with Prometheus and Grafana

## Overview
This project demonstrates a monitoring setup using Prometheus and Grafana.

## Architecture
Node Exporter → Prometheus → Grafana

## Features
- Real-time system monitoring
- CPU, memory, disk metrics
- Dashboard visualization

## Tools Used
- Prometheus
- Grafana
- Node Exporter
- Docker

## How to Run
docker compose up -d
